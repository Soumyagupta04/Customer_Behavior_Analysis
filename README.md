📊 Customer Shopping Behavior Analysis
📌 Overview
This project presents an end-to-end data analytics workflow to analyze customer shopping behavior. It covers data loading, exploratory data analysis (EDA), data cleaning, SQL-based analysis using PostgreSQL, and visualization through an interactive Power BI dashboard. Business insights are summarized in a structured report and presentation.

📁 Dataset
The dataset contains customer shopping and transactional information, including:
-Customer demographics
-Purchase history
-Product categories
-Discounts and loyalty indicators
-Spending patterns
The data was cleaned and prepared for analysis using Python before being stored in a PostgreSQL database.

🛠 Tools & Technologies
1.Python (Pandas, NumPy, Matplotlib, Seaborn)
2.PostgreSQL (data storage & SQL queries)
3.SQLAlchemy (Python–PostgreSQL connection)
4.Power BI (dashboard & visual analysis)
5.Gamma (presentation creation)
6.Jupyter Notebook (analysis & documentation)

🔍 Project Steps
1.Loaded the dataset using Python
2.Performed Exploratory Data Analysis (EDA)
3.Cleaned and transformed the data
4.Stored cleaned data in a PostgreSQL database
5.Ran SQL queries for business insights
6.Built an interactive Power BI dashboard
7.Created an analytical report
8.Designed a presentation using Gamma

📊 Dashboard
The Power BI dashboard highlights:
-Customer demographics and segmentation
-Spending and purchase behavior trends
-Product category performance
-Impact of discounts and loyalty programs
The dashboard is designed to support data-driven business decisions.

📈 Key Results & Insights
-Identified high-value customer segments
-Analyzed spending behavior across demographics
-Evaluated product category performance
-Highlighted factors influencing repeat purchases
These insights can help optimize marketing strategies and improve customer retention.
