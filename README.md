# 📊 Customer Shopping Behavior Analysis

---

## 📌 Overview
This project presents an **end-to-end data analytics workflow** to analyze customer shopping behavior.  
It covers data loading, exploratory data analysis (EDA), data cleaning, SQL-based analysis using **PostgreSQL**, and visualization through an interactive **Power BI dashboard**.

Business insights are summarized in a structured report and presentation.

---

## 📁 Dataset
The dataset contains customer shopping and transactional information, including:

- Customer demographics  
- Purchase history  
- Product categories  
- Discounts and loyalty indicators  
- Spending patterns  

The data was cleaned and prepared using **Python**, then stored in a **PostgreSQL database**.

---

## 🛠 Tools & Technologies
- **Python** (Pandas, NumPy, Matplotlib, Seaborn)  
- **PostgreSQL** (data storage & SQL queries)  
- **SQLAlchemy** (Python–PostgreSQL connection)  
- **Power BI** (dashboard & visualization)  
- **Gamma** (presentation creation)  
- **Jupyter Notebook** (analysis & documentation)  

---

## 🔍 Project Workflow
1. Loaded the dataset using Python  
2. Performed Exploratory Data Analysis (EDA)  
3. Cleaned and transformed the data  
4. Stored cleaned data in PostgreSQL  
5. Executed SQL queries for business insights  
6. Built an interactive Power BI dashboard  
7. Created an analytical report  
8. Designed a presentation using Gamma  

---

## 📊 Dashboard
The Power BI dashboard highlights:

- Customer demographics and segmentation  
- Spending and purchase behavior trends  
- Product category performance  
- Impact of discounts and loyalty programs  

The dashboard is designed to support **data-driven decision-making**.

---

## 📈 Key Insights
- Identified high-value customer segments  
- Analyzed spending behavior across demographics  
- Evaluated product category performance  
- Highlighted factors influencing repeat purchases  

These insights help optimize **marketing strategies** and improve **customer retention**.
